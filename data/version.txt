Retrieval of the active player and all their quests.

Added the ability to set different colors to highlight varying values.
Added the ability to set different colors based on remaining quest time.
Added quest sorting by a selected key.
Added multi-level sorting with an unlimited number of keys.
Added the ability to toggle sorting order alphabetically.
Added a filter to display only selected sectors.
Added shortcuts for easy switching of sorting and filtering.
