[Module] Quest
[Access] SCUM.db

[1.0] Alpha
[Logic]

Automatic detection of the SinglePlayer database.
Optimized database access through indexing.
Retrieval of the active player and all their quests.
Retrieval of all required quest parameters.
Retrieval of the time values used in quests.

[Display]

Each line has its own font size, type, and color settings.
Option to format the header into multiple lines with custom data.
Option to format quests into multiple lines with custom data.
Added settings for formatting quest countdown timers — from hours down to seconds.
Added simulation between quest time updates from the database.
Added the ability to set different colors to highlight varying values.
Added the ability to set different colors based on remaining quest time.
Added quest sorting by a selected key.
Added multi-level sorting with an unlimited number of keys.
Added the ability to toggle sorting order alphabetically.
Added a filter to display only selected sectors.
Added shortcuts for easy switching of sorting and filtering.
